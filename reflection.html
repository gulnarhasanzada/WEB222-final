Responsive design and web technologies

Responsive design is a way of developing applications so that the way that app is displayed depends on the screen size of the device using it. Designers use relative measurement units, like % and apply media queries to adapt the web application to browsers size.

In the early days of the internet web sites only were used in PC and laptops. Although, developers used fixed size it was not an issue. As long as it looked good on a laptop, the website would look almost the same in any other monitor.

However, arrival of mobile computing has changed everything. A website that looks nice on laptop would look messy on mobile. Large objects, fonts divs wouldn’t fit on small sizes, so it required scrolling, zooming in and out all the time. So, web sites were not user-friendly on mobile. Therefore, developers started to use adaptive design. They started to build 6 different and fixed sized GUI-s for different device sizes (320, 480, 760, 960, 1200, and 1600 pixels). Based on device size browsers would define which design to display. However, it was very expensive solution and devices with non-standard size didn’t have an optimal view.

Use of relative sizes has solved many problems in modern responsive web design. For example, width: 100% would work for every device. For browser window that has 1280px size it would mean 1280px and for mobile it would mean 320px. So, one design can fit for every device. Fluid grid systems, fluid images and media queries are three main principles used in building of modern responsive apps. There are many frameworks that can help designers to build fully responsive websites. For instance, Bootstrap is on of the most popular ones and it’s free.

Building web apps for different devices

Through this research I learned:

  • How evolution of devices and the internet has changed the understanding of web design and lead to the development of responsive design
  • Two different methods used to make websites user-friendly in every device: Adaptive and responsive design
  • Similarities and differences between adaptive and responsive design
  • Some frameworks that support responsive design
